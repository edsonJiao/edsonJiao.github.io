改变世界的人

发表于 2017-09-23 | 分类于 梦想

最近这几天，因为跟好朋友出现了一点分歧，我觉得是时候放慢一下脚步，回头看看自己曾经走过的路、踩过的坑了。也正好可以为自己的未来好好的做一次思考了。

19


不知不觉我已经在世界上存在了19年了，时间很快。一转眼，就已经成年了，可能下一次感觉时间流逝的时候，我已经结婚生子了。
时间真的好快啊，19年就这样的过去了，在宿舍还会偶尔跟舍友吹一下自己小学、初中的牛逼，那时候也没感觉时间流逝的多么快，可当真的静下心来回首往事的时候，会发实现时间过得真的好快，
一直感觉高中结束还在昨天，可现在猛然一想，一年多已经过去了。
这19年来，得到了很多，同时也失去了很多，踩了无数的坑，走了很多弯路。
很久之前，一直以为自己很聪明，因为眼界的问题，自大自满，直到看到了外面的世界，才真正的懂得了人外有人。
19年来，我以自己的方式，用自己的个性活着，为自己的天真吃了很多亏，也为自己的个性收获了很多别人得不到的喜悦。

19年来最后悔的事就是浪费了很多的时间，浪费了自己的天赋，在那个时间充足的时候，我选择了挥霍时间，却没做一些真正有意义的事情。
浪费的时间真的是回不来了，只能在未来的日子里用1.5倍的努力进行弥补。


至于性格



我的性格比较偏执，不是很喜欢说话，也不喜欢主动去讨好别人。总是被长辈们说没有前途，不会处理关系。我真的是很反感这种观点。我追求的是自由，一种无拘无束，没有束缚的生活，不管是对人，还是
对事，无愧于心就好。不想被别人的决定所左右。讨好巴结本来就不是我该做的事。就想孔子所说：君子之交淡如水。平平淡淡、自由自在就好，无所谓什么讨好取悦，当一种感情、一种关系真的变成了这样，那
一个人活得跟一条狗没什么区别。所幸，我遇到的朋友，那些真正的好朋友也获得很真实，跟他们在一起，我也没有那么沉默内向，因为我们三观相同，所展现的是真实的自我，嬉笑打骂，都很自然。这是我所追求的
关系，幸运的是，我认识了很多。前几天，一个好朋友跟我说：我没有像你那样的追求，不过我也有自己的梦想，希望你不要看不起我。其实，他说这话的时候，我真的很佩服他，我真的没有看不起他，人各有志，
只不过追求的东西不同，对我来说衡量一个人是不是朋友的标准，除去本身的素质，就是追求，知道自己想要的是什么，为之努力即可，追求可以不同，但一定要有，也要为之奋斗。我可以跟一个满怀理想，并为之努力的
乞丐交朋友，也不会跟一个整天高谈梦想却无所行动的人打一声招呼。


至于梦想



梦想这个词近几年来已经被用烂了，不管什么事都称之为梦想，抛开现实性，我的梦想是创立一家真真正正的中国的科技公司。
就目前来说，中国最强的几家互联网公司，无非是BAT，B已经没落，曾经的百度是跟Google一样的靠搜索引擎发家，可是后来却相距甚远，Google研究无人驾驶、人工智能、研究癌症的治理，而百度呢？还在卖外卖，抢打车市场。
我用我自己很浅的简介来评定，BAT，算不上真正的科技公司，而是三家由三个极其优秀的商人领导的以科技为生产力的专注于利润的公司。而Google、Facebook、微软呢？这才是真真正正的科技公司，他们将眼光放在未来，
执着于对未来的探索，利润也许只是他们的副产品（我是这么认为的，可能因为自己的目光短浅）。这也许跟他们公司创立有关，Google创始人拉里·佩奇和谢尔盖·布林、Facebook创始人扎克伯格、微软创始人比尔盖茨这些人本身就是极客，
本身追求的是技术，而不是利润。极客跟商人是不同的，一个追求未来、追求极致，一个追求现实、追求实际，就像《社交网络》里，扎克伯格跟爱德华多的分歧一样，这也可能是BAT与Google等公司差别的原因吧。
最主要的原因，还是因为中国的国情，中国是一个人情为主的国家（并没有说不好），这在一定程度上耗费了BAT的精力，他们把更多的时间花在人际关系的处理上，而不是技术的钻研。

而我所要做的就是创立一所真正的科技公司，在他们的眼中是异类也好，疯子也好，都无所谓，甚至连成功与否都无所谓。只有这样，中国的科技才能真正的发展，而不是至于表面。我渴望成功，无关财富，只是想成为像乔布斯、扎克伯格、比尔盖茨一般能够改变世界的人，
至少对中国做出一点改变和影响。我也不惧怕失败，人生如白驹过隙，转瞬即逝，希望自己的失败会聚集更多像我一样的异类、疯子。那个时候，也许也算一种成功了。


我会按照自己的方式努力下去，之前可以向现实妥协，这次不行了，我们的时代来了。
